#TraktTV
[![Build Status](https://travis-ci.org/pixis/TraktTV.svg?branch=master)](https://travis-ci.org/pixis/TraktTV)

Android app that notifies you when the next episode of your trakt watchlist airs.

Uses:
- OAuth2
- RxJava
- Realm
- Retrofit2
